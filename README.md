## Run it locally

1. Clone this repository to your computer:

   ```bash
   git clone https://github.com/sreesaiogetiammini/FinalProjectCS6010
   cd FinalProjectCS6010
   ```

2. compile the program with CMake:

   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. run the program in your terminal

   `./blackjack`
